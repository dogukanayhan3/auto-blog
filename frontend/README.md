# Auto-Blog Frontend

React frontend for the auto-generated blog.

## Setup
```bash
npm install
```

## Environment Variables

Create a `.env` file:
```
REACT_APP_API_URL=http://localhost:5000/api
```

## Running
```bash
# Development
npm start

# Build for production
npm run build
```

## Features

- Browse all articles
- Read individual articles
- Responsive design
- Clean, modern UI
